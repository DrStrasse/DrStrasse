# GRM Handcuffs

Отдельный аддон наручников для Garry's Mod с интеграцией в вашу систему фракций.

## Установка

Папку `grm_handcuffs` положить в:

```text
garrysmod/addons/
```

## Класс оружия

```text
grm_handcuffs
```

Его можно добавить во фракционную выдачу оружия через ваш `/weapons_admin`.

## Управление

- ЛКМ по игроку — надеть наручники.
- ЛКМ по задержанному — снять наручники.
- ПКМ по задержанному — начать/остановить ведение.
- R по задержанному — надеть/снять кляп.
- ALT + R по задержанному — надеть/снять повязку на глаза.
- E по задержанному — удерживать для снятия наручников другим игроком, если доступ разрешён.
- E по транспорту, когда ведёте задержанного — посадить его на пассажирское сиденье.
- E по транспорту с задержанным внутри — вытащить задержанного и начать вести.

Задержанный сам себя освободить не может.

## Что есть

- Ограничение доступа по фракциям, ролям и отделам.
- Отдельный патч доступа `zz_grm_handcuffs_access_patch.lua`.
- Drag/ведение задержанного.
- Кляп: блокирует чат и voice.
- Повязка: затемняет экран.
- HUD подсказки.
- Верёвка между задержанным и ведущим.
- Защита от смены оружия в наручниках.
- Блок прыжка, спринта, подбора оружия/предметов, входа в транспорт.

## Выдача для теста

Серверная консоль или superadmin:

```text
grm_cuffs_give <ник/SteamID/SteamID64>
```

Если аргумент не указан и команда вводится игроком — выдаёт себе.

## Интеграция с существующими фракциями

Аддон теперь автоматически читает данные из вашей существующей системы:

```text
garrysmod/data/factions.json
garrysmod/data/fw_faction_extras.json
garrysmod/data/default_weapons.json
```

Логика доступа:

1. Если глобальная таблица `Factions` уже загружена вашим аддоном фракций — используется она.
2. Если она ещё не загружена — аддон читает `data/factions.json`.
3. Для оружия читаются `Weapons`, `RoleWeapons`, `DepartmentWeapons` из `fw_faction_extras.json`.
4. Если игроку через `/weapons_admin` выдано оружие `grm_handcuffs`, он получает право им пользоваться.

Проверка:

```text
grm_cuffs_debug_access <ник/SteamID/SteamID64>
```

Перезагрузка кэша:

```text
grm_cuffs_reload_factions
```

## Доступ

Настройка доступа делается отдельным файлом. По умолчанию доступ берётся из вашей существующей выдачи оружия `/weapons_admin`: если фракции/рангу/отделу выдан `grm_handcuffs`, значит пользоваться наручниками можно.

```text
zz_grm_handcuffs_access_patch.lua
```

Положить сюда:

```text
garrysmod/addons/grm_handcuffs/lua/autorun/zz_grm_handcuffs_access_patch.lua
```

## Подсказки и поза рук

При наведении на игрока с оружием `grm_handcuffs` показываются подсказки:

```text
ЛКМ — надеть/снять наручники
ПКМ — взять и вести / отпустить
R — кляп
ALT + R — повязка на глаза
```

У задержанного игрока клиент визуально ставит руки за спину. Поза обновлена: теперь используется принудительная BuildBonePositions-поза, которая сводит кисти/предплечья за спину, плюс за спиной рисуется короткая стяжка. Если нужно отключить позу:

```lua
GRM.Handcuffs.Config.EnableBehindBackPose = false
```

Самостоятельное освобождение отключено параметром:

```lua
GRM.Handcuffs.Config.CanSelfRelease = false
```

## Посадка в транспорт

Поддерживается:

```text
обычный Garry's Mod транспорт
simfphys
LVS
```

Как использовать:

1. Наденьте наручники на игрока.
2. Возьмите его вести через `ПКМ`.
3. Наведитесь на транспорт.
4. Нажмите `E`.

Система ищет свободное пассажирское место и не использует первое/водительское место.

Чтобы вытащить задержанного:

1. Наведитесь на транспорт с задержанным внутри.
2. Нажмите `E`.

Задержанный не может сам выйти из транспорта.

Настройки:

```lua
GRM.Handcuffs.Config.VehicleUseDistance = 170
GRM.Handcuffs.Config.VehicleSeatSearchRadius = 420
GRM.Handcuffs.Config.VehicleExitOffset = 80
GRM.Handcuffs.Config.VehicleAllowDriverSeat = false
```

## Оружие при сковывании

При надевании наручников весь текущий loadout игрока временно сохраняется и убирается.
Игрок получает только служебное оружие состояния `grm_cuffed` без модели.

Настройки:

```lua
GRM.Handcuffs.Config.StripWeaponsOnCuff = true
GRM.Handcuffs.Config.RestoreWeaponsOnUncuff = true
GRM.Handcuffs.Config.EnforceNoWeaponsWhileCuffed = true
```

Если какая-то система сервера попытается выдать задержанному физган/оружие во время задержания, таймер автоматически уберёт его снова.
